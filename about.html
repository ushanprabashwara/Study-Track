About StudyTrack

StudyTrack is a fully client-side Student Assignment Tracker built with HTML, CSS, JavaScript, Bootstrap 5, FullCalendar, and Chart.js. All data lives in your browser's Local Storage — no backend, no accounts, no tracking.

Features

  • Dashboard with progress, charts, and deadline alerts
  • Full Course and Assignment CRUD
  • FullCalendar monthly view with priority colors
  • Search, filter, and sort assignments
  • Reports with pie & bar charts
  • CSV export, backup & restore
  • Responsive — desktop, tablet, mobile

Tech Stack

  • HTML5 / CSS3 / ES6
  • Bootstrap 5 & Bootstrap Icons
  • FullCalendar 6
  • Chart.js 4
  • Local Storage

Deploy on GitHub Pages

  1. Push this folder to a repo.
  2. Settings → Pages → Deploy from main.
  3. Open dashboard.html.
